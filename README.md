## Responsive Flexbox Layout

A simple responsive web design project built using **HTML** and **CSS Flexbox**. This project demonstrates how a three-column layout adapts to different screen sizes using media queries.

## Features

- Responsive layout using Flexbox  
- Three columns on desktop view  
- Stacked (vertical) layout on smaller screens  
- Clean and minimal design  
- Easy to understand structure for beginners  

## Technologies Used

- HTML5  
- CSS3 (Flexbox & Media Queries)  

## Responsive Behavior

- **Desktop / Large Screens:**  
  Displays three boxes side by side  

- **Mobile / Small Screens (≤ 740px):**  
  Boxes stack vertically for better readability
